\doxysubsection{Detailed Description}
\hypertarget{README.md_autotoc_md1}{}\doxysubsection{\texorpdfstring{Описание проекта}{Описание проекта}}\label{README.md_autotoc_md1}
Это клиент-\/серверное приложение для вычисления выпуклой оболочки множества точек на плоскости. Система состоит из\+:


\begin{DoxyEnumerate}
\item {\bfseries{Сервера}} -\/ принимает запросы от клиентов, вычисляет выпуклую оболочку и возвращает результат
\item {\bfseries{Клиента}} -\/ отправляет точки на сервер и сравнивает результат с локальным вычислением
\item {\bfseries{Массового тестера}} -\/ проверяет сервер на множестве случайных данных
\item {\bfseries{Визуализатора}} -\/ создает графическое представление тестовых случаев
\end{DoxyEnumerate}\hypertarget{README.md_autotoc_md2}{}\doxysubsection{\texorpdfstring{Требования}{Требования}}\label{README.md_autotoc_md2}

\begin{DoxyItemize}
\item Windows ОС
\item Компилятор с поддержкой C++17
\item Python 3.\+x (для визуализации)
\item Библиотеки Python\+: matplotlib
\item Библиотека Winsock2 (включена в Windows)
